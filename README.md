# Space-Inviders

## Опис гри

Space Invaders - класична аркадна гра, де гравець керує космічним кораблем і захищає Землю від вторгнення прибульців. Гравець може рухати корабель вліво-вправо та стріляти лазерними пострілами вгору. Вороги рухаються строєм, стріляють вниз і поступово опускаються до низу екрану. Мета - знищити всіх прибульців, перш ніж вони досягнуть поверхні планети, та набрати максимальну кількість очок.

**Реалізовані додаткові фічі:**

1. **Пауза гри**: натискання клавіші `P` ставить гру на паузу і відновлює її з того ж місця. На екрані з'являється повідомлення "PAUSED", рух корабля, пришельців та пуль зупиняється.
2. **Вибухи пришельців**: при попаданні гравецьської кулі в прибульця з'являється анімація вибуху з частинок (5–10), які розлітаються і поступово зникають. Кожен ворог має власний ефект вибуху, при бажанні можна додати звук через Web Audio API.
3. **Анімований фон зі зірками**: на фоні з’являються зірки різного розміру та яскравості, які повільно рухаються вниз, створюючи ефект космічного простору. Зірки з’являються зверху після того, як виходять за межі екрану.

## Інструкції як запустити

1. Завантажте файл `space-invaders.html`.
2. Відкрийте його у сучасному веб-браузері (Chrome, Firefox, Edge, Safari).
3. Керування:
   - `ArrowLeft` або `A` - рух вліво
   - `ArrowRight` або `D` - рух вправо
   - `Space` - стріляти
   - `Enter` - перезапуск гри після закінчення
   - `P` - пауза / відновлення гри

Гра працює повністю в одному HTML-файлі, без додаткових налаштувань або бібліотек.

## Опис процесу створення

1. Завантажено технічне завдання з `AGENTS.md`.
2. AI (Copilot) допоміг створити базову версію гри в одному HTML-файлі з Canvas, керуванням, пришельцями, стрільбою, рахунком та життями.
3. Гра протестована в браузері, перевірено плавність анімацій та правильність механік.
4. Додано додаткові фічі:
   - **Пауза гри** (`P`) із зупинкою всіх елементів та повідомленням на екрані.
   - **Вибухи пришельців** із частинками, які поступово зникають.
   - **Анімований фон зі зірками** з рухом і мерехтінням для більш живого космічного ефекту.
5. Гра оптимізована для плавного відображення на різних роздільних здатностях та тестована на Chrome і Firefox.
6. Весь код, включно з CSS і JavaScript, вбудований у один HTML-файл, що відповідає вимогам AGENTS.md.
